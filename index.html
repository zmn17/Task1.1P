I'm a full stack developer with a passion for creating user-friendly and efficient web applications.

Here's what I've done so far!

TripMate

It's a travel API backend service that allows user to search through AirBnb places/homes and book them for holiday or trips. The good feature about this is that the user can create trip, he/she can add his/her friends to this trip. The app then tracks the expenses of the trip, the fees of the place/home booked from AirBnb and generate a report to the user. This API service is built with ASP.NET Core in C#, PostgreSQL for data storage and management, it scrapes data from Airbnb website for the Airbnb home/place to be booked for the trip.

Weather App

It's a full stack weather app that allows user to search for weather of any city in the world. The app is built with React for frontend and NodeJS for backend.

SportAPI

It's a sports API backend service that allows user to search through sports events by scraping data from sport websites and storing them in database. The app is built with ASP.NET Core in C#, PostgreSQL for data storage and management.
